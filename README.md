Object-Storage-Browser
==================

Object-Storage-Browser is a file browser and uploader for object storage services using an S3-compliant API including Amazon S3 itself and DreamObjects. Fully HTML5 based,
no server needed.

Object-Storage-Browser is simply a browser-based client to your S3-compliant storage enabling you to download and upload files in buckets.

Setup: Customizing Object-Storage-Browser look and feel
===================================================
Edit css/branding.css.
